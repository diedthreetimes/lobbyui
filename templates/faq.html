Introduction

UnTap is a multiplayer CCG deck tester. Decks are built and play-tested from various formats against other opponents online. Our unique deck builder uses API searches to build your deck ready to play-test simple and mess free. Everything is in the browser no downloads or plugins required.

Deck Building

Each CCG type has its own set deck field options, but for the most part all cards are added in the same way.

Basic Input

DECK INPUT
4 Mystic Enchant
4 Flaming Sword
2 Dragon Breath
EXPLAINED

Each Line is prefixed with a quanity followed by the name of the card. This applies to all card input areas (Main/Secondary deck, Sideboard, Alt Cards)

Importing a Deck

There are two methods for importing decks uploading a deck file or fetching from a deck URL. Uploading Deck files supports txt dec l5d o8d file types.
Fetching decks from URL works by pulling a deck list down from a URL, if the page has a deck list on it, UnTap will do its best to import it. Not all URLs are supported, please make sure that the deck page is public and login is required to access the page.


Custom Cards

Since UnTap uses remote URLs for all its card images you can by pass the card search API and manually add a custom image URL for a card. Custom cards will flag the deck as containing custom cards to prevent people changing card text, or atleast making you cautious to the know that the card is custom.

DECK INPUT
4 Token Frog^http://url.img/tokenfrog.jpg
4 Red Monkey^http://custom.art/monkey.jpg
2 Rules card^http://rules.com/card.jpg
EXPLAINED

Each line is still prefixed with a quantity, but appending ^ then a URL to the card name will bypass the API search
